# Helperbot

¡Bienvenido a Helperbot! Tu asistente virtual inteligente.

## 📋 Descripción

Helperbot es una aplicación web diseñada para ayudarte con tus tareas y responder tus preguntas de manera rápida y eficiente.

## 🚀 Características

- Interfaz intuitiva y moderna
- Diseño responsive para todos los dispositivos
- Fácil de usar y navegar

## 📁 Estructura del Proyecto

```
Helperbot/
├── index.html      # Página principal
├── styles.css      # Estilos de la aplicación
├── script.js       # Lógica de la aplicación
└── README.md       # Este archivo
```

## 💻 Cómo Usar

1. Descarga o clona el repositorio
2. Abre el archivo `index.html` en tu navegador
3. ¡Comienza a usar Helperbot!

## 🛠️ Tecnologías Utilizadas

- **HTML5**: Estructura de la aplicación
- **CSS3**: Estilos y diseño responsive
- **JavaScript**: Interactividad y funcionalidades

## 📝 Notas

Esta es una versión inicial de Helperbot. Se añadirán más características y funcionalidades en futuras actualizaciones.
